# fireworks

vue3.0 + canvas 实现烟花助力2022高考

## 第一步：安装node.js
[node.js 官网](https://nodejs.org/zh-cn/)
## 第二步：安装依赖库

```shell
cd fireworks
npm install
```

## 第三步：运行代码
```shell
npm run dev
```
